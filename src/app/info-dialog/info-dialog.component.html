About this App

This app is saving all times between check-in and check-out in the localStorage.

Then it is calculating the total time of this times.

Then it calculates a Start- and End-time (start is set to 8:00) which adds the time for the break (30min after 6h and 45min after 9h).

If only two times (check-in and check-out) then a break time is removed from total time.
